# IT Access Request Procedure

## How to Request Access

All access requests must go through the IT Service Portal.

## Types of Access

### Standard Access (Auto-Approved)

New employees automatically receive:
- Corporate email
- Slack workspace
- Shared drives (relevant department)
- VPN access

### Requested Access (Manager Approval)

- Additional shared drives
- Production system access
- Database read access
- Third-party service accounts

### Elevated Access (Director + Security Approval)

- Database write access
- Production deployment rights
- Admin privileges on any system
- Access to other departments' data

## Request Process

1. Log in to IT Service Portal
2. Select "New Access Request"
3. Choose system/resource from catalog
4. Provide business justification
5. Submit for approval

## Approval Timeline

- Standard: Immediate
- Requested: 1-2 business days
- Elevated: 3-5 business days

## Revocation

Access is automatically revoked:
- Upon employment termination
- After 90 days of inactivity (elevated access)
- When role changes (access review required)

## Security

Never share your credentials. Each access is logged and audited.
Violations may result in disciplinary action.
