# IT Equipment Procurement

## Standard Equipment

New employees receive standard equipment package:
- Laptop (MacBook Pro or ThinkPad T-series)
- External monitor (27")
- Keyboard and mouse
- Headset for calls
- Webcam (if laptop cam insufficient)

## Requesting Additional Equipment

1. Submit request via IT Service Portal
2. Provide business justification
3. Manager approval required
4. IT evaluates and sources equipment

## Replacement Policy

- Laptops: Every 3 years or upon failure
- Monitors/peripherals: Upon failure only
- Battery replacement: When capacity < 70%

## Personal Equipment

Using personal devices for work is discouraged.
If necessary, enroll device in MDM (Mobile Device Management).
Company compensates up to $50/month for personal device use.

## Software

All software must be approved by IT before installation.
Pirated or unlicensed software is strictly prohibited.
Standard software is pre-installed on all company devices.
